# 📊 Análisis de Presión Tributaria, Empleo y Recaudación en Argentina

Este proyecto forma parte de una diplomatura en análisis de datos y tiene como objetivo estudiar la relación entre:

- Presión tributaria
- Recaudación
- Mercado laboral
- Evolución económica anual

El análisis se desarrolla utilizando **Python, Pandas y visualización de datos**, integrando diferentes datasets económicos.

---

# 🧰 Herramientas utilizadas

- Python
- Pandas
- Matplotlib / Plotly
- Jupyter Notebook
- Análisis exploratorio de datos (EDA)

# 🎯 Objetivo del proyecto

Analizar cómo se relacionan:

- la carga tributaria
- el nivel de empleo
- la recaudación fiscal

para identificar patrones económicos y posibles impactos en la actividad económica.

---

# 📌 Conclusiones principales

- La presión tributaria presenta una tendencia creciente a lo largo del período analizado.
- El empleo formal muestra una caída significativa durante el año 2020 debido a la pandemia.
- La recuperación posterior del empleo no siempre ocurre al mismo ritmo que la recaudación o la presión fiscal.
- El análisis integrado permite observar cómo los cambios en política fiscal y contexto económico impactan en el mercado laboral.
